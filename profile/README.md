🧭 About Natrix

Natrix Client is a Minecraft PvP client organization focused on delivering a high-performance, customizable, and smooth gameplay experience for 1.8.9 PvP.

We enhance how the game feels and performs, without changing core gameplay.

✨ Core Features
  ⚡ FPS Optimization – smoother gameplay in fights
  🎮 Responsive Input – reduced delay and lag spikes
  🎨 Custom Animations – 1.7 / 1.8 style support
  ❤️ HUD Enhancements – clean and customizable UI
  🌌 Visual Customization – sky, effects, enchant glint
  🧍 Movement Animations – sneak, block hit, item held
  🧰 Modular System – toggle everything easily
